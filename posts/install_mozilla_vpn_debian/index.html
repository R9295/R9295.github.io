How to install MozillaVPN Client on Debian 11 (Bullseye)

Introduction

MozillaVPN’s README is already quite good, but is lacking a few Debian specific packages.

MozillaVPN is built with Qt, so it’s moments like these when you appreciate good distributions cause compiling Qt is a nightmare

Approach

Install and setup Qt5

sudo apt install qtdeclarative5-dev \
libqt5networkauth5-dev \
libqt5websockets5-dev \
libqt5charts5-dev \
qtbase5-dev \
qtchooser \
qt5-qmake \
qtbase5-dev-tools \
qttools5-dev \
qml-module-qtcharts \
qml-module-qtgraphicaleffects \
qml-module-qtquick-controls \
qml-module-qtquick-controls2 \
qml-module-qtquick-extras \
qml-module-qtquick-layouts \
qml-module-qtquick-window2 \
qml-module-qtquick2 \
qml-module-qtqml-models2 qml-module-qtqml

vim ~/.bashrc
export PATH="/usr/lib/qt5/bin:$PATH" # Add this line

source ~/.bashrc

Install MozillaVPN dependencies

# I assume you have go and python3 installed
sudo apt install wireguard \
wireguard-tools \
libpolkit-gobject-1-dev

Clone the repo and init submodule

git clone https://github.com/mozilla-mobile/mozilla-vpn-client
cd mozilla-vpn-client/
git submodule init
git submodule update

Create an env for some python scripts that need to be run to setup the build

python3 -m venv env
source ./env/bin/activate

Setup the build

pip3 install -r requirements.txt

# glean
./scripts/generate_glean.py
# translations
./scripts/importLanguages.py
# Bake shaders (qt6 only)
sh ./scripts/bake_shaders.sh

Build

qmake CONFIG+=production
make -j$(nproc)
sudo make install

Run!

mozillavpn

debian mozillavpn
